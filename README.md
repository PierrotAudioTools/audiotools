# AudioTools

Site statique prêt pour GitHub Pages pour présenter et distribuer des outils.

## Structure

- `index.html` : structure de la page
- `styles.css` : identité visuelle et responsive
- `script.js` : données des outils et rendu des cartes
- `.nojekyll` : compatibilité GitHub Pages sans traitement Jekyll

## Modifier les outils

Édite le tableau `tools` dans `script.js` pour changer :

- le nom
- la catégorie
- la description
- les formats / plateformes
- le lien de téléchargement
- le lien secondaire

## Déployer sur GitHub Pages

1. Push le contenu dans un repo GitHub
2. Active GitHub Pages sur la branche voulue
3. Serre le site depuis la racine du repo

## À personnaliser

- Remplacer `contact@example.com`
- Remplacer les URLs `https://github.com/`
- Adapter les outils à tes vrais produits
